University WebApp

Student project built with Spring Boot, Spring Data JPA, Thymeleaf, and PostgreSQL.

The application provides CRUD features for university entities and will later include a monthly schedule view.

Database tables (core)

  • Students
  • Teachers
  • Groups
  • Courses
  • Lessons
  • Users / Roles (security)
  • Join tables for many-to-many relations (e.g., student_courses)
